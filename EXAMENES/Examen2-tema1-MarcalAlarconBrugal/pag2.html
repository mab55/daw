Etiqueta BR

La etiquet br, se usa para hacer un salto de línea.

Ejemplo:

Hola
esta línea sigue debajo.

Etiqueta H1

La etiquet h1, se usa para hacer un título principal.

Ejemplo:

Esto es un texto normal

Esto es un título

Etiqueta code

La etiquet code, se usa para decirle al navegador que estamos escribiendo un código.

Ejemplo:

Posición relativa: position: relative

Etiqueta sup

La etiquet sup, se usa para colocar un texto en la parte superior.

Ejemplo:

Esto es un texto normal Esto es un texto en la parte de arriba

Etiqueta sub

La etiquet sub, se usa para colocar un texto en la parte inferior.

Ejemplo:

Esto es un texto normal Esto es un texto en la parte de inferior

Etiqueta strong

La etiquet strong, se usa para destacar un contenido.

Ejemplo:

Esto es un texto normal Esto es un texto destacado
