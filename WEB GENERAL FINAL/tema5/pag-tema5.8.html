5.8 Técnicas Errores Comunes

¿Qué son los Errores Comunes?

Los errores comunes, son errores que cometemos a la hora de crear un sitio web. Aquí se agrupan todos los errores que no se pueden clasificar en algún tipo de técnica.

Errores Comunes más vistos

1. No se cumple el criterio de cumplimiento 1.3.1 debido al uso de cambios en la presentación del texto para transmitir información sin usar el marcado o el texto adecuados.

Este error ocurre cuando no seguimos un patrón de marcado semántico apropiado.

Ejemplo



2. 2. Fallo del Criterio de Conformidad 4.1.1 debido a valores duplicados de ID.

Este error ocurre cuando repetimos dos o más id. Cada elemento solo puede tener un id que tiene que ser único.

Ejemplo



3. Fracaso del Criterio de Conformidad 1.4.3 y 1.4.6 debido al uso de imágenes de fondo que no brindan suficiente contraste con el texto de primer plano (o imágenes de texto)

Este error ocurre cuando escribimos un texto de un color que no contrasta lo suficiente con el fondo.

Ejemplo



4. No se cumple el criterio de cumplimiento 1.4.8 debido al uso de texto justificado (alineado con los márgenes izquierdo y derecho)

Este error ocurre cuando le aplicamos a un texto la propiedad Justify, esta propiedad hace que el texto aparezca justificado y sea menos legible al contener espacios entre palabras diferentes.

Ejemplo



5. Fracaso del Criterio de Conformidad 2.2.2 debido al uso del elemento blink

Este error ocurre cuando usamos la etiqueta blink sobre un texto. Esta etiqueta hace que un texto parpadee constantemente. No debemos usar esta etiqueta, ya que puede dañar a la vista.
