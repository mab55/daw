9.1 DETAILS Y SUMMARY

La etiqueta <summary> define un encabezado visible para el elemento <details>. Haciendo clic en el <summery> se muestra/oculta el contenido

Epcot Center

Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.

<details>
<summary>Epcot Center</summary>
<p>Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.</p>
</details>

9.2 DIÁLOGOS EMERGENTES y VENTANAS MODALES

El elemento <dialog> facilita la creación de cuadros de diálogo emergentes y modales en una página web. Con el atributo “open” mostramos el popup.

This is an open dialog window
<dialog open>This is an open dialog window </dialog>

9.3 CONTENIDO EDITABLE

Con este atributo, establecemos si un párrafo es editable o no. Simplemente colocamos contenteditable="true" y ese párrafo es editable.

This is an editable paragraph.

<p contenteditable="true">This is an editable paragraph.</p>

9.4 ATRIBUTO HIDDEN

Con este atributo, conseguimos mostrar/ocultar un contenido. Simplemente añadimos la palabra “hidden” y listo.

This paragraph should be hidden.

<p hidden>This paragraph should be hidden.</p>
