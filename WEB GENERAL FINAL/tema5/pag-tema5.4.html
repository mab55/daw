5.4 Técnicas HTML

¿Qué son las técnicas HTML?

Las técnicas generales son técnicas establecidas con el uso de los recursos del lenguaje HTML.

Técnicas HTML más usadas

1. Proporcionar un título, usando la etiqueta Title

Esta técnica consiste en asignarle un título a cada una de nuestras páginas usando la etiqueta title.

Ejemplo



2. Uso de atributos alt en elementos img

Cuando una imagen contiene palabras que son importantes para comprender el contenido, el texto alternativo debe incluir esas palabras. Esto permitirá que el texto alternativo tenga la misma función en la página que la imagen. Tenga en cuenta que no describe necesariamente las características visuales de la imagen en sí, pero debe transmitir el mismo significado que la imagen.

Ejemplo



3. Uso de ol, ul y dl para listas o grupos de enlaces

Esta técnica consiste en usar las etiquetas de listas (ol, ul y dl) para crear nuestras listas dentro de la página.

Ejemplo



4. Uso del atributo de idioma en el elemento HTML

Esta técnica consiste en asignarle a cada página el idioma de la web. Así el navegador es capaz de interpretar la página en ese idioma.

Ejemplo



5. Uso de atributos de autocompletar HTML 5.2

Esta técnica consiste en asignarle la propiedad autocompletar, para que los navegadores sean capaces de autocompletar un campo de un formulario.

Ejemplo



6. Asegurarse de que los elementos no contengan atributos duplicados

Consiste en tener mucho cuidado a la hora de crear los atributos de una etiqueta. Es muy fácil equivocarse y repetir atributos. Eso hace gastar recursos innecesarios al navegador de nuestro cliente.

Ejemplo



7. Uso de h1-h6 para identificar encabezados

A la hora de crear un sitio web es muy importante su ESTRUCTURA, manteniendo una jerarquía entre las diferentes secciones de nuestra web. Usando h1 para el título principal, h2 para el título secundario...

Ejemplo



8. Asegurarse de que las etiquetas de apertura y cierre se utilicen de acuerdo con la especificación.

Esta técnica consiste en comprobar que siempre que escribamos una etiqueta la abrimos y cerramos correctamente.

Ejemplo



9. Uso de meta actualización para crear una redirección instantánea del lado del cliente

El objetivo de esta técnica es permitir redireccionamientos en el lado del cliente sin confundir al usuario. Los redireccionamientos se implementan preferiblemente en el lado del servidor, pero los autores no siempre tienen control sobre las tecnologías del lado del servidor.

Ejemplo



10. Usando el elemento rubí

El objetivo de esta técnica es utilizar la anotación ruby ​​para proporcionar información sobre la pronunciación y el significado de un texto en el que el significado está determinado por la pronunciación. Hay muchos idiomas en los que una serie de texto puede significar diferentes cosas dependiendo de cómo se pronuncia el texto. Esto es común en los idiomas de Asia oriental, así como en hebreo, árabe y otros idiomas; también ocurre en inglés y otros idiomas de Europa occidental. Ruby Annotation permite al autor anotar un "texto base", proporcionando una guía para la pronunciación y, en algunos casos, también una definición.
