1.6 LAS TABLAS

¿Qué es una tabla?

Una tabla no es otra cosa más que un medio de organizar datos en filas y columnas. Como muchas otras estructuras de HTML, las tablas son construidas utilizando elementos.

Elementos de una tabla

Para crear una tabla en HTML debemos de usar las siguientes etiquetas:

table

Con esta etiqueta establecemos el contenedor principal). Es muy importante abrir esta etiqueta al empezar la tabla y cerrarla cuando terminemos la tabla.

tr

Representando a las filas contenedoras de las celdas.

td

Representando a las celdas de la tabla.

Estructura de las tablas

A la hora de crear tablas, es importante mantener una buena estructura, para ello deberemos de usar las siguientes etiquetas:

thead

Establecemos la cabecera de la tabla.

tbody

Establecemos el cuerpo de la tabla.

tfoot

Establecemos el pie de la tabla.

Propiedades de las celdas

Cuando creamos una tabla, a las celdas les podemos asignar las siguientes prioridades:

RowSpan

Permitir que una celda ocupe más de una fila.

ColSpan

Permite que una celda ocupe más de una columna.
