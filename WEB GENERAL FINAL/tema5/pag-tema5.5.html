5.5 Técnicas CSS

¿Qué son las técnicas CSS?

Las técnicas CSS son técnicas establecidas para mejorar la accesibilidad y la percepción de la información en forma visual.

Técnicas CSS más usadas

1. Usar porcentaje para tamaños de fuente

Esta técnica consiste en usar PORCENTAJES en vez de pixeles para los textos. De forma que depende del tamaño de la pantalla, el texto ocupa un tamaño u otro.

2. Uso de CSS width, max-width y flexbox para ajustar etiquetas y entradas

Esta técnica consiste en establecer un tamaño en PORCENTAJE, pero estableciendo a los elementos un tamaño máximo/mínimo.

3. Especificar el espaciado de línea en CSS

Esta técnica consiste en establecer un espaciado entre las líneas de un texto.

4. Usar CSS para ocultar una parte del texto del enlace

Esta técnica consiste en establecer un tamaño en PORCENTAJE, pero estableciendo a los elementos un tamaño máximo/mínimo.
